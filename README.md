## 使用

> Step 1.先在 build.gradle(Project:XXXX) 的 repositories 添加:

	allprojects {
		repositories {
			...
			maven { url "https://jitpack.io" }
		}
	}
	
> Step 2. 然后在 build.gradle(Module:app) 的 dependencies 添加:

	dependencies {
	       implementation 'com.github.oyo775881:ShopAISdk:+'
	}
